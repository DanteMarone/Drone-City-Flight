# House Veranda

## Overview
A cozy brick house with a covered veranda, planter boxes, rain barrel, and trellis accents. The porch includes an animated ceiling fan for extra life.

## Visuals
- Brick-walled main volume with a pyramidal roof.
- Concrete porch platform with steps and painted posts.
- External elements: planter box greenery, rain barrel, and a wooden trellis.
- Porch ceiling fan spins gently via the `update(dt)` loop.

## Key Parameters
- `width` (default: 11): Overall width of the main body.
- `height` (default: 6): Wall height of the main body.
- `depth` (default: 9): Overall depth of the main body.

## Dependencies
- Extends `BaseEntity`.
- Uses `TextureGenerator.createBrick` and `TextureGenerator.createConcrete` for materials.
