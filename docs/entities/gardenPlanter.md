# Garden Planter

## Overview
A raised planter box filled with stylized flowers and leafy stems. It brings garden-scale greenery into plazas and parks while adding gentle motion through subtle plant sway.

## Visuals
- Wooden planter body with trim and four support legs.
- Dark soil inset inside the box.
- Procedural plant clusters: tapered stems, conical leaves, and ringed petals around a glowing flower center.

## Key Parameters
- `length`: Overall planter length. Defaults to a random value around 2.1–2.7.
- `width`: Overall planter width. Defaults to a random value around 0.9–1.2.
- `height`: Overall planter height. Defaults to a random value around 0.55–0.7.

## Dependencies
- Extends `BaseEntity`.
- Registered via `EntityRegistry` in `src/world/entities/gardenPlanter.js`.
