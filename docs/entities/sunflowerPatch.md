# Sunflower Patch

## Overview
A clustered patch of tall sunflowers anchored in a soil mound. The heads gently sway, bringing a lively garden vibe and visual variety to outdoor spaces.

## Visuals
- Cylindrical soil mound with a thin rim collar.
- Multiple sunflower stems with layered cone leaves.
- Sunflower heads built from a central disk and radial cone petals, tilted for character.

## Key Parameters
- `radius`: Radius of the soil mound. Defaults to a random value around 0.9–1.3.
- `height`: Height of the soil mound. Defaults to a random value around 0.25–0.35.
- `count`: Number of sunflower stems. Defaults to 4–6.

## Dependencies
- Extends `BaseEntity`.
- Registered via `EntityRegistry` in `src/world/entities/sunflowerPatch.js`.
