# Flowering Trellis

## Overview
A decorative trellis planter wrapped with climbing vines and glowing blossoms. It adds vertical greenery to gardens and gently animates with swaying vines and pulsing flowers.

## Visuals
- Low wooden planter filled with soil.
- Tall trellis frame with crossbars and lattice slats.
- Procedural vine tubes dotted with conical leaves and soft-glow blossoms.

## Key Parameters
- `width`: Trellis width. Defaults to a random value around 1.6–2.0.
- `height`: Trellis height. Defaults to a random value around 2.2–2.8.
- `depth`: Trellis depth. Defaults to a random value around 0.35–0.5.

## Dependencies
- Extends `BaseEntity`.
- Registered via `EntityRegistry` in `src/world/entities/floweringTrellis.js`.
