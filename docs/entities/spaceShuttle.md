# Space Shuttle

## Overview
The Space Shuttle is a grounded orbital-style craft prop with pulsing engine glow, adding futuristic aerospace flavor to the city.

## Visuals
- **Fuselage:** Rotated cylinder with a cone nose, wrapped in a striped CanvasTexture.
- **Cockpit:** Tinted sphere scaled into an oval canopy.
- **Wings:** Wide box wings with dark wingtip panels.
- **Tail:** Vertical fin and tail block for a shuttle silhouette.
- **Engines:** Triple rear engine pods with emissive glow that pulses over time.
- **Boosters:** Side-mounted cylindrical boosters for added bulk.

## Key Parameters
- `length`: Overall body length (default: `8.5`).
- `bodyRadius`: Main fuselage radius (default: `1.1`).
- `wingSpan`: Total wing span (default: `5.4`).

## Dependencies
- Extends `BaseEntity`.
- Uses `THREE.CanvasTexture` for the hull livery.
