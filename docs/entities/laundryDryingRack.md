# Laundry Drying Rack

## Overview
Apartment-friendly drying rack prop with hanging laundry that gently sways over time. Designed to add residential life to courtyards, balconies, and rooftops.

## Visuals
- Four metal posts with cross braces and a top support bar.
- Multiple clotheslines stretch across the rack.
- Procedurally generated fabric textures on hanging cloth planes.
- A small wicker basket at the base for extra domestic detail.

## Key Parameters
- `width`: Overall width of the rack (default randomized).
- `depth`: Overall depth of the rack (default randomized).
- `height`: Overall height of the rack (default randomized).
- `lineCount`: Number of clotheslines (default `3`).
- `frameColor`: Hex color for the metal frame.

## Dependencies
- Extends `BaseEntity`.
- Uses Three.js primitives and CanvasTexture for fabric patterns.
