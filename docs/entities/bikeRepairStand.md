# Bike Repair Stand

## Overview
The Bike Repair Stand is an urban maintenance prop that provides a clamp station, pump, and tool panel for cyclists in the city.

## Visuals
- Concrete base slab with a metal post and crossbar clamp arms.
- Tool panel with a generated control texture and a glowing status screen.
- Pump cylinder, handle, and a looping hose/nozzle assembly.

## Key Parameters
- `baseWidth`, `baseDepth`, `baseHeight`: Size of the concrete platform.
- `postHeight`: Height of the main support post.
- `accentColor`: Optional emissive tint for the status screen.

## Dependencies
- Extends `BaseEntity`.
- Uses `TextureGenerator.createSidewalkBlank` for the base texture.
